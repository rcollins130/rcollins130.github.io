OAT: Optical Aircraft Tracking

Stanford ME354 Final Project. I track airplanes from my back window using a software-defined radio, a DSLR, and computer vision code written in MATLAB. The "deliverable" dataset was a comparison of aircraft speed estimates from the camera to speeds reported by the aircraft over ADS-B. Source code can be found on Github here: OAT Repository
